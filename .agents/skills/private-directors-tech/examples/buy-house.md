# 示例：买房决策

```text
/private-directors 我该不该买房？预算 120 万美元，家庭年收入 35 万美元，有一个孩子，未来 3 年可能搬州。
```

## 推荐顾问

- 房产与居住董事
- 资本与投资董事
- 风险与反脆弱董事
- 价值观与人生设计董事
- 关系与伦理董事
- 执行与运营董事
- 红队董事

## 重点澄清

1. 首付、利率、月供和现金储备是多少？
2. 未来 3 年搬州概率多高？
3. 买房主要目标是稳定、投资、学区、身份认同还是避免 FOMO？
4. 如果房价下跌 20% 且需要搬家，能否承受？
5. 伴侣对稳定性和流动性的偏好是什么？
