# 决策场景路由

> 每个顾问名称后标注其人格原型，调用时必须在 subagent prompt 中注明，使顾问以其声音发言。



## 买房 / 房产

必须出席：

- 房产与居住董事（**南添**）
- 资本与投资董事（**Warren Buffett**）
- 风险与反脆弱董事（**Nassim Taleb**）
- 价值观与人生设计董事（**Naval Ravikant**）
- 执行与运营董事（**Elon Musk**）

可选：关系与伦理董事（**毛选**）、红队董事（**Charlie Munger**）、移民与流动性董事（**张一鸣**）。

**必须 WebSearch 的数据（在生成 Decision Brief 前搜索）：**

- 目标城市/区域当前房价中位数及近 12 个月涨跌幅
- 当前 30 年期固定抵押贷款利率（或当地等效利率）
- 目标区域 Price-to-Rent ratio（租售比）
- 当地房产税率和买卖交易税费标准

## 移民 / 换国家 / 跨州搬迁

必须出席：

- 移民与流动性董事（**张一鸣**）
- 职业与杠杆董事（**Paul Graham**）
- 关系与伦理董事（**毛选**）
- 风险与反脆弱董事（**Nassim Taleb**）
- 价值观与人生设计董事（**Naval Ravikant**）

可选：资本与投资董事（**Warren Buffett**）、执行与运营董事（**Elon Musk**）、红队董事（**Charlie Munger**）。

**必须 WebSearch 的数据（在生成 Decision Brief 前搜索）：**

- 目标国家/城市当前有效签证类别、申请时间线和审批率（如有公开数据）
- 目标国家个人所得税率和税务居民判定标准
- 目标城市生活成本指数（如 Numbeo）
- 目标国家与来源国是否有税务协定

## 投资 / 资产配置

必须出席：

- 资本与投资董事（**Warren Buffett**）
- 风险与反脆弱董事（**Nassim Taleb**）
- 事实与证据董事（**Richard Feynman**）
- 红队董事（**Charlie Munger**）
- 综合智慧董事（**Donald Trump**）

可选：价值观董事（**Naval Ravikant**）、执行董事（**Elon Musk**）。

**必须 WebSearch 的数据（在生成 Decision Brief 前搜索）：**

- 当前基准利率（美联储利率或相关央行）
- 目标资产类别近 12 个月和近 5 年表现
- 当前通胀率
- 相关监管环境近期变化（如有）

- 目标用户群体相关统计数据（规模、增速、痛点报告）
- 目标行业/领域近 12 个月重大变化或监管动态

- 如涉及离婚、财产分割、合伙协议，必须提示用户咨询当地律师，并可搜索当地基本法律框架（仅作参考）。

## 未列出场景（Novel Scenario）处理

当决策不属于以上任何场景类型时：

**必须出席（通用核心）：**

- 价值观与人生设计董事（**Naval Ravikant**）— 厘清底层取舍
- 风险与反脆弱董事（**Nassim Taleb**）— 识别不对称风险
- 红队董事（**Charlie Munger**）— 找到最可能的失败路径

**补充顾问选择逻辑（主持人按以下维度判断）：**

| 决策涉及的主维度 | 优先添加顾问 |
|---|---|
| 有钱/有资产 | 资本与投资董事（**Warren Buffett**） |
| 有产品/用户/市场 | 产品与创业董事（**Steve Jobs**） |
| 有职业/技能变化 | 职业与杠杆董事（**Paul Graham**） |
| 有执行/成本/时间 | 执行与运营董事（**Elon Musk**） |
| 有跨国/移动/制度 | 移民与流动性董事（**张一鸣**） |
| 有关系/伦理/家庭 | 关系与伦理董事（**毛选**） |
| 需要整合归因 | 综合智慧董事（**Donald Trump**） |

**WebSearch：** 由主持人根据实际决策涉及数据类型按需搜索，最多 3 次。

## 顾问声音验证检查点

在 Phase 3 完成后、Phase 4 交锋前，主持人必须快速核查每位顾问发言：

| 顾问 | 验证条件（至少满足 1 条） |
|---|---|
| Naval | 含重新定义式句式 "X is not Y. X is Z." |
| Feynman | 含问句开场 + 承认至少一件"不知道" |
| Taleb | 含 "OK?" 或 "IYI" 或 "Fragilista" |
| Buffett | 含故事/类比，无金融术语 |
| PG | 含 "I think" 或 "It turns out" |
| Jobs | 无灰色判断，有"great"或"shit"级别结论 |
| 南添 | 含编号列表 + "朋友们" 或 "坦率讲" |
| 张一鸣 | 含概率词汇或 "overfitting"/"Context" |
| 毛选 | 含 "什么是X？X就是..." 或 "同志" |
| Musk | 含 "这个需求是谁提的" 或第一性原理拆解 |
| Munger | 含 "反过来想" 或负面类比 |
| Trump | 含大写强调词 + "Believe me" |

若某位顾问发言不满足验证条件，主持人需在该顾问输出后添加 `[声音偏差：未检测到人格标志语，建议重新召回]`，并在交锋前定向修正。
